
交差判定  
以下の連立方程式を解く

- Rt + Drt*x = Ra + Dra*x
- Ct + Dct*x = Ca + Dca*x

(Drt - Dra)*x = Ra - Rt
x = (Ra - Rt) / (Drt - Dra)

(Drt - Dra) = 0の時は同じ方向の移動or別軸同士の移動
同じ方向の移動の場合は以下の２通りとなる
- 同じ位置を移動し続ける
- 別の位置を移動し続ける

以下の条件で合流地点が見つかる
0 < x <= w
wは1セクション

移動ケース
  U D L R  
U 1 2 3 3
D 2 1 3 3
L 3 3 1 2
R 3 3 2 1

- case 1: 同じ位置を移動し続ける
- case 2: 同じ座標軸で移動する
- case 3: 別の座標軸で移動する
